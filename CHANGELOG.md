# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025-05-22

### Added

- System tray application with history dialog and settings
- Floating bubble overlay (single HWND) with stack badge, animations, pop effect
- Clipboard monitoring via `AddClipboardFormatListener`
- SQLite history and JSON settings
- Global hotkey, launch at startup, drag-to-paste with window highlight
- Image thumbnail pipeline (200×150 max) with full PNG stored separately
